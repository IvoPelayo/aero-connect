# AeroConnect — Prueba Práctica

Aplicación de búsqueda y reserva de vuelos.  
Tienes acceso a todo el código base. Lee el código antes de empezar.

---

## Arrancar el proyecto

```bash
npm install
npm start
```

- App: `http://localhost:4200`  
- API mock: `http://localhost:8882/api`
